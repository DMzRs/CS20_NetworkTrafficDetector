# CS20_NetworkTrafficDetector

## Before running anything, Make sure to set up sa ENV first!

### How to run VENV
#### To set up:
python -m venv venv
#### To run env:
.\venv\Scripts\activate
#### To stop env:
deactivate

### Before running:
run this command "pip install -r requirements.txt"

### How to run Streamlit
streamlit run app.py
